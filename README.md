# common-config

프로젝트 내 사용하는 공통된 eslint, prettier를 정리해둔 패키지 입니다.

'우리 팀을 위한 ESLint, Prettier 공유 컨피그 만들어보기' 를 참고했습니다.

---

## 사용법

### eslint 적용

1) example-react 내 eslint 관련 수정사항을 작성합니다. 
2)  `yarn example-react lint` 를 실행해 eslint 수정사항이 발생한 것을 확인합니다.


### prettier 적용
1) example-* 내 prettier 관련 수정 사항을 작성합니다.
2) `yarn example-react prettier`를 실행해 prettier 관련 수정된 사항을 확인합니다.
